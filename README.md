# UGC Outfit Generator (Next.js)

This repository contains a Next.js app for generating UGC outfit images using AI Studio (Gemini).

## Features
- Upload product image + optional model image
- Full control panel: gender, hair style, hair color, pose, lighting, vibe, etc.
- Backend API uses a prompt (your spec) to call AI Studio / Gemini
- Results returned as base64 images (display + download)
